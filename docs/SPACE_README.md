# Wolfgang-LM

**A Historical Tiny Language Model for Goethe-Persona Preservation**

Chat with a digital recreation of Johann Wolfgang von Goethe. The model responds in authentic 18th-century German style.

## About
- **Architecture**: ~75M Parameter Llama-style Transformer
- **Training**: Pretrained on Deutsches Textarchiv (1600-1900) + finetuned on synthetic dialogues
- **Context**: 512 tokens

## Resources
*   **Model**: [dominik-reiner/wolfgang-lm-75M-instruct-v1](https://huggingface.co/dominik-reiner/wolfgang-lm-75M-instruct-v1)
*   **Dataset**: [dominik-reiner/wolfgang-lm-dataset](https://huggingface.co/datasets/dominik-reiner/wolfgang-lm-dataset)
*   **GitHub**: [dominik-reiner/wolfgang-lm](https://github.com/dominik-reiner/wolfgang-lm)

## Usage
Simply type your message in German and the model will respond as Goethe.

## Disclaimer
WOLFGANG-LM is an AI system. Interactions are simulated. Outputs may contain historical biases or inaccuracies.
